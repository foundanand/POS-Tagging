# POS TAGGER 

POS Tagger

## Installation

1. Clone the repository.
    ```shell
    git clone https://github.com/foundanand/POS-Tagging.git
    ```
2. Install the dependencies.
    ```shell
    pip install -r requirements.txt
    ```

## Usage

1. Run the main.py file.
    ```shell
    python main.py
    ```
2. Follow the instructions given on code
